# MakuTweaker for Linux 🐧 (Alpha)

⚠️ **Ограничение Alpha-версии:** На данный момент после закрытия и перезахода в приложение состояние переключателей (чекеров) в интерфейсе сбрасывается на дефолтное. Сами твики при этом **успешно применяются в системе**, интерфейс пока просто не умеет считывать их текущий статус обратно. Это будет исправлено в ближайших версиях.

---

Утилита для полной настройки, оптимизации и персонализации Linux-систем. За основу проекта взят и полностью переосмыслен популярный инструмент [MakuTweaker для Windows](https://github.com). 

Проект написан на чистом **C** с использованием графического фреймворка **GTK4**. Текущая версия находится на стадии **альфа-тестирования**, активно дорабатывается и в будущем будет расширяться.

---

## 🚀 Основные возможности
* **Управление процессами**: встроенный интерактивный монитор системных процессов.
* **Файловый менеджер**: легковесный инструмент навигации и работы с файлами.
* **Инструменты debloat**: безопасное удаление ненужного системного мусора и неиспользуемых компонентов.
* **Тонкая настройка**: оптимизация производительности и параметров безопасности системы.
* **Персонализация**: управление темами оформления интерфейса.

---

## 📦 Зависимости и требования

Для корректной сборки и графической настройки интерфейса программы в вашей системе должны быть установлены необходимые библиотеки разработки.

### Зависимости для сборки (Fedora)
```bash
sudo dnf groupinstall "Development Tools" && sudo dnf install gtk4-devel wayland-devel glib2-devel pkgconf-pkg-config
```

### Зависимости для сборки (Ubuntu / Debian)
```bash
sudo apt update && sudo apt install build-essential libgtk-4-dev libwayland-dev libglib2.0-dev pkg-config
```

### Для настройки внешнего вида GTK (Темы, шрифты, иконки)
Чтобы кастомные темы оформления корректно подтягивались в систему, рекомендуется установить графические утилиты конфигурации:
```bash
# Для Fedora
sudo dnf install gnome-tweaks dconf-editor

# Для Ubuntu / Debian
sudo apt install gnome-tweaks dconf-editor
```

⚠️ **Важное примечание по кастомизации:** На текущем этапе разработки функционал динамической смены цвета интерфейса поддерживается только внутри тайлинговых оконных менеджеров **Sway** и **Hyprland**.

## 🛠️ Сборка и установка

Поскольку проект находится в стадии альфа-версии, бинарные пакеты отсутствуют, и утилита собирается вручную из исходного кода.

1. Скомпилируйте проект с помощью утилиты `make`:
   ```bash
   make
   ```

2. Для применения низкоуровневых системных твиков программе необходим фоновый бэкенд. Скопируйте скомпилированный модуль `tweaker_backend` в системный каталог исполняемых файлов:
   ```bash
   sudo cp tweaker_backend /usr/local/bin/
   ```

3. Запустите графическую оболочку твикера:
   ```bash
   ./makutweaker_gui
   ```
