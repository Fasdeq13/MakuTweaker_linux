# MakuTweaker for Linux

Комплексный системный твикер для Linux-систем, написанный на чистом языке C (стандарт C11) с использованием базового инструментария GTK4. Разработан эксклюзивно для Wayland-сессий.

## Особенности архитектуры
- **Ноль тяжелых зависимостей:** Не использует Libadwaita, Qt или обертки. Только чистый GTK4 C-API.
- **Прямое взаимодействие с ядром:** Данные о процессах, памяти и процессоре парсятся вручную из виртуальных файловых систем `/proc` и `/sys`.
- **Разделение привилегий:** - `makutweaker_gui` выполняется с правами обычного пользователя.
  - `makutweaker_backend` вызывается изолированно через `pkexec` для записи изменений в защищенные файлы ядра и выполнения административных команд.
- **Встроенная локализация:** Ручная интернационализация (RU/EN) на структурах данных без использования `gettext`.
